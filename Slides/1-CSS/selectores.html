Etiqueta HTML

Podemos asignar un estilo a todas las etiquetas de un determinado tipo.

h1, h2, h3, h4, h5, h6 {
    font-family: 'Oswald', sans-serif;
}

H1 con la fuente 'Oswald'

Clases

Cuando declaramos un estilo mediante el nombre de una clase debemos tener en cuenta lo siguiente:

  • La declaración deben comenzar con un punto.
  • Solo se permiten letras, números, guiones o el guion bajo.
  • Debe comenzar por una letra despues del punto.
  • CSS distingue entre mayúsculas y minúsculas.
.box-red {
    box-shadow: 0 0 5px #000;
    background-color: #C0392B;
}
.box-red

Identificadores

Los estilos asignados mediante identificadores deben comenzar mediante el simbolo #. Ventajas y desventajas de utilizar identificadores para declarar estilos:

Ventajas:

  • Nos facilita la seleccion del elemento mediante JavaScript.
  • Nos permite navegar directamente a ese identificador por medio de un link <a>.

Desventajas:

  • Asigna al estilo una prioridad alta, lo que dificulta que se apliquen otros estilos definidos mediante otros selectores.
  • No permite su reutilización, es decir, un identificador es único y solo se aplican los estilos a él mismo.
#circle-blue {
    border-radius: 50%;
    background-color: #0170BA;
}
#circle-blue

Selector universal *

El selector universal nos permite aplicar un estilo determinado a todos los elementos HTML.

* {
    display: inline-block;
    width: 100px;
    height: 100px;
    line-height: 100px;                         
    margin: 10px;              
    text-align: center;                             
    color: #FFF;
    background-color: #0170BA;

Descendientes

Podemos definir una clase a partir de su posición dentro de la jerarquia del DOM. Hay que tener en cuenta que la prioridad de ese elemento será la suma de todos las etiquetas indicadas en el estilo. 

.headline h1 {
    font-size: 1.5rem;
    font-weight: bold;
    font-family: monospace;
}

Atributos

Podemos seleccionar un elemento a partir de sus atributos:

  • [atributo]: Seleccionamos aquellas etiquetas que tienen declarado un determinado atributo.
  • [atributo = valor]: Selecciona aquellas etiquetas cuyo atributo tienen un determinado valor.
  • [atributo ^= subcadena]: Aquellos atributos cuyo valor empieza por una determinada subcadena.
  • [atributo $= subcadena]: Aquellos atributos cuyo valor termina por una determinada subcadena.
  • [atributo *= subcadena]: Aquellos atributos que contienen una determinada subcadena.
a[href] {
    text-decoration: underline;
}

a[href="http://www.google.com"] {
    color: #C0392B;
}

a[href^="https://"] {
    text-transform: uppercase;
}

a[href$=".es"] {
    color: #000;
}

a[href*="avanade"]
    text-decoration: none;
}

  • https://www.google.com
  • http://www.google.es
  • http://www.avanade.com

Seleccionando hermanos

Podemos aplicar un estilo a un elemento en función de sus nodos hermanos. Los estilos se aplican en función de los hermanos definidos previamente, no se puede aplicar en función de los hermanos que le preceden..

  • h1 + h2: Aplica el estilo al elemento h2 que está declarada justo a continuación de un elemento h1.
  • h1 ~ p: Aplica el estilo a todas las etiquetas p que están definidas despues de una etiqueta h1 pero no es necesario que hayan sido declaradas justo a continuación.
h1 + h2 {
    padding-left: 2rem;
    font-size: 2em;
    font-style: italic;
}

h1 ~ p {
    padding: 1em 0;
    color: #0170BA;
}

Pseudo-clases y Pseudo-elementos

Pseudo-clases vs Pseudo-elementos:

  • Las Pseudo-clases se utilizan para definir estados especiales de un elemento, se declaran mediante dos puntos :
  • Los Pseudo-elementos nos permite seleccionar partes especificas de un determinado elemento, se declaran mediante ::

::before y ::after

Los pseudo-elementos ::before y ::after solo se pueden aplicar a etiquetas que permiten añadir contenido (div, span, p, etc.). No se permite a otro tipo de etiquetas tales como input. A estos pseudo-elementos les acompaña la propiedad content que determina el contenido que se va a mostrar y que siempre debe estar definida.

p::before{
    content:'Antes';                         
    display:block;
    position:relative;
    left:-100%;
    background-color:#0170BA;
    color:#FFF;
}

p::after{
    content:'Despues';                         
    display:block;
    position:relative;
    right:-100%;
    background-color:#C0392B;
    color:#FFF;
}

Elemento

::selection

Con este pseudo-elemento solo se pueden aplicar las propiedades background-color y color.

p::selection {                                     
    color:#FFF;
    background-color:#C0392B;
}
