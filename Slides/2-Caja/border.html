border-top, border-right, border-bottom y border-left

Estas propiedades permiten establecer los bordes de cada lado de forma independiente.

.box {
    border-top: 5px dotted blue;
    border-right: 3px dashed green;
    border-bottom: 5px solid gray;
    border-left: 6px solid red;
}

border-width, border-style y border-color

.box {
    border-width: 3px;
    border-style: solid;
    border-color: #F4792B;
}

border-radius

.box-1 {
    border-radius: 5px;
}

.box-2 {
    border-radius: 50%;
}

.box-3{
    border-radius: 10px/20px;
}

.box-4 {
    border-top-left-radius: 20px;
    border-bottom-right-radius: 20px;
}

Atajos para los bordes

Podemos establecer con una propiedad un valor independiente para cada lado.

.box {
    border-width: 5px 2px 5px 4px;
    border-style: dotted solid dashed ridge;
    border-color: red green blue black;
    border-radius: 5px 10px 2px 20px;
}
