Display

  • Modelo de caja
  • Display

En HTML hay dos tipos de cajas, block o inline. Una etiqueta de tipo block crea un salto antes y despues de ella. Mientras que una etiqueta de tipo inline no produce estos saltos, apareciendo las etiquetas una al lado de la otra.

Las etiquetas de tipo inline tienen otra distinción; en ellas las propiedades de la caja se pueden aplicar al rango horizontal, pero no en vertical. Es decir, una etiqueta de tipo inline, por ejemplo <span> permitirá que le especifiquemos un margin-left o margin-right, pero las propiedades margin-top y margin-bottom serán descartadas. A estas etiquetas de tipo inline tampoco se les puede indicar una altura.

Esta distinción entre tipos de caja podemos sobreescribirla mediante la propiedad display cuyos valores puedes ser:

  • block: Aplica el comportamiento de bloque, añadiendo saltos antes y despues.
  • inline: Evita los saltos antes y despues de la etiqueta.
  • inline-block: A veces queremos que una etiqueta siga siendo inline pero a la vez poder modificar sus propiedades verticales (margin-top, margin-bottom, height, padding-bottom y padding-top)
  • none: Oculta el elemento y ademas no ocupa su espacio dentro de la navegaciónd el DOM. A diferencia de la propiedad visibility.
  • table: Mediante este display se permite que el contenido del elemento se muestra como una <table>. Permitiendo establecer filas y columnas dentro del contenido.
  • flex: Permite establecer un contenedor flexible. Detallado en el apartado Layout: Caja flexible.

La propiedad display tiene otro valores posibles, sin embargo estos son los utilizados comunmente.

div {
    border: 2px solid blue;
}
           
div.inline {         
    display: inline;
    background-color: blue;
    color: white;
}

span {
    border: 2px solid red;
}             

span.block {
    display: block;
    background-color: blue;
    color: white;
}
