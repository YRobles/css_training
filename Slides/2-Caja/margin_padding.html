margin-top, margin-right, margin-bottom y margin-left

Hay que tener en cuenta que cuando se aplica un valor en porcentajes tanto a lo ancho como a lo alto, este porcentaje es calculado en función de la anchura de la página.

.box-px {
    margin-top: 20px;
    margin-left: 40px;
}

.box-em {
    margin-top: 2em;
    margin-left: 1.5em;
}

.box-percentage {
    margin-left: 20%;
}

margin

Mediante este atajo podemos aplicar un margen a cada lado de una sola vez.

.box-1 {
    margin: 5px 10px 20px 15px;
}

.box-2 {
    margin: 10px 30px 10px;
}

.box-3 {
    margin: 20px 10px;
}

.box-2

Colapso de margenes

El colapso de margenes se producen cuando dos elementos situados verticalmente uno encima del otro, cada uno aplica un margen vertical. En este caso, la separación vertical entre ambos elementos será el valor del margen máximo y no la suma de ambos márgenes.

.box-top {
    margin-bottom: 30px;
}

.box-bottom {
    margin-top: 20px;
}
.box-top
.box-bottom

padding-top, padding-right, padding-bottom y padding-left

Al igual que con los márgenes, cuando se aplica un valor en porcentajes tanto a lo ancho como a lo alto, este porcentaje es calculado en función de la anchura de la página.

.box-px {
    padding-top: 20px;
    padding-left: 40px;
}

.box-em {
    padding-bottom: 2em;
    padding-right: 1.5em;
}

.box-percentage {
    padding-left: 20%;
}
.box-px
.box-em
.box-percentage

padding

Atajo para establecer todos los padding con una sola llamada.

.box-1 {
    padding: 5px 10px 20px 15px;
}

.box-2 {
    padding: 10px 30px 10px;
}

.box-3 {
    padding: 20px 10px;
}
.box-1
.box-2
.box-3
