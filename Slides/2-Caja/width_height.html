width y height

Cuando asignamos la anchura y altura de un elemento debemos tener en cuenta lo siguiente:

  • Si asignamos un ancho y alto en pixeles, el contenido que no quepa en el contenedor saldra de él. Para gestionar este comportamiento trabajaremos con la propiedad overflow.
  • Siempre que sea posible, debemos realizar un diseño fluido o responsive, en cuyo caso los anchos y altos serán establecidos mediante porcentajes.
  • Si asignamos un porcentaje a la altura, este porcentaje será calculado en función de la anchura del contenedor padre si su altura no ha sido definida.
  • La altura y anchura total de la caja es calculada en funcion de la suma del contenido, el padding, el margen y los bordes. Si queremos modificar este comportamiento utilizaremos la propiedad box-sizing.
.box {
    width: 50%;
    height: 100px;
}

Establecer límites al ancho y al alto

Podemos establecer un ancho mínimo y un ancho máximo:

  • max-width: Establece el ancho máximo que puede llegar a alcanzar una caja.
  • min-width: Ancho mínimo que puede tener la caja.
  • max-height: Establece la altura máxima a la que puede llegar la caja.
  • min-height: Altura mínima a la que se establecerá la caja.

Calcular el tamaño

El ancho y el alto total de la caja se calcula sumando su contenido, su padding, sus bordes y su margen.

.box {
    width: 200px;
    height: 100px;
    padding: 10px 20px;                         
    margin: 20px 30px;
    border: 5px solid #AAA;
}
Ancho: 250px
Alto: 130px

box-sizing

Mediante esta propiedad le estamos indicando como debe calcular el ancho y alto de la caja y que propiedades debe tener en cuenta para calcularlo:

  • content-box: (Valor por defecto) El tamaño lo calcula segun el contenido. No tiene en cuenta el padding o el borde.
  • border-box: Calcula la altura y anchura incluyendo el contenido + borde + padding.
.box-content {
    box-sizing: content-box;
}
                    
.box-border {
    box-sizing: border-box;
}

overflow

La propiedad overflow determina como debe actuar la caja cuando el contenido sobresale de ella. Normalmente tras aplicarle un tamaño de ancho y alto fijo.

  • visible: por defecto, como si la propiedad no existiera
  • scroll: añade un scroll al elemento haga falta o no
  • auto: muestra el scroll cuando el contenido sobrepasa la anchura o altura
  • hidden: oculta el contenido que sale fuera del contenedor

La propiedad overflow también se puede aplicar independiente para cada eje mediante las propiedades overflow-x y overflow-y.

.box-visible {
    overflow: visible;
}

.box-scroll {
    overflow: scroll;
}

.box-auto {
    overflow: auto;
}

.box-hidden {
    overflow: hidden;
}
