Caja flexible

  • Layout
  • Caja flexible

El modelo de caja flexible o flexbox nos permite adaptar nuestros diseños de layout de tal modo que se adapten de una forma predecible segun indiquemos y de una manera flexible a todo tipo de resoluciones.

display: flex

Esta propiedad nos permite indicar a un contenedor que todos sus hijos deberan comportarse como elementos flexibles. Los elementos hijos suelen ir acompañados de la propiedad flex.

.flex-container {
    display: flex;
}

.flex {
    flex: 1;
    height: 200px;
}

flex

La propiedad flex indica la flexibilidad de ese elemento con respecto al resto de elementos hermanos. Por defecto su valor es flex: 0 1 auto, siendo cada valor lo siguiente:

  • El primer valor indica el ancho relativo del elemento con respecto a los demás.
  • El segundo valor flex-shrink indica como de estrecho puede llegar a ser el elemento.
  • El ultimo valor es el flex-basis e indica el ancho inicial para un elemento.
.flex-container {
    display: flex;
}

.flex-1 {
    flex: 1;
}

.flex-2 {
    flex: 2;
}

.flex-3 {
    flex: 3;
}

flex-direction

Esta propiedad indica como debe disponer los elementos, si como filas o como columnas.

  • row: (Valor por defecto) Muestra los elementos uno al lado del otro de izquierda a derecha.
  • row-reverse: Uno al lado del otro pero de derecha a izquierda.
  • column: Muestra los elementos uno encima de otro.
  • column-reverse: Invierte verticalmente el orden de los elementos.
.flex-container {
    display: flex;
    flex-direction: column;
}

flex-wrap

Controla si los elementos salta a una nueva fila o columna cuando no caben en el contenedor.

  • nowrap: (Valor por defecto) Los elementos estarán en la misma fila o columna.
  • wrap: Los elementos que no caben en el contenedor pasan a una nueva fila o columna.
  • wrap-reverse: Igual que wrap pero en orden invertido.
.flex-container {
    display: flex;
    flex-wrap: wrap-reverse;
}

.flex-200 {
    flex: 1 0 200px;
}

.flex-300 {
    flex: 1 0 300px;
}

.flex-400 {
    flex: 1 0 400px;
}

flex-flow

Esta propiedad es un atajo que nos permite indicar las propiedades flex-direction y flex-wrap juntas

.flex-container {
    display: flex;
    flex-flow: row nowrap;
}

.flex-200 {
    flex: 1 0 200px;
}

.flex-300 {
    flex: 1 0 300px;
}

.flex-400 {
    flex: 1 0 400px;
}

justify-content

Permite indicar como se distribuye el contenido cuando hay espacio sobrante.

  • flex-start: Alinea los elementos a la izquierda de la fila. Si la propiedad flex-direction está puesta como row-reverse será a la derecha.
  • flex-end: Alinea los elementos a la derecha.
  • center: Coloca el elemento en el centro del contenedor.
  • space-between: Reparte el espacio entre todos los elementos de una forma equitativa.
  • space-around: El espacio sobrante es repartido equitativamente a ambos lados.
.flex-container {
    display: flex;
    justify-content: space-around;
}

.flex {
    width: 50px;
    height: 100px;
}

align-items

La propiedad align-items es útil cuando tenemos elementos flexibles de diferentes alturas y queremos controlar donde ubicarlos verticalmente.

  • stretch: (Valor por defecto) Estira los elementos para que ocupen la misma altura.
  • flex-start: Alinea los elementos en la parte superior del contenedor.
  • flex-end: Alinea los elementos en la parte baja del contenedor.
  • center: Coloca los elementos centrados verticalmente.
  • baseline: Alinea la línea base del primer elemento con el resto.
.flex-container {
    display: flex;
    align-items: center;
    height: 200px;
}

.flex-100 {
    flex: 1;
    height: 100px;
}

.flex-50 {
    flex: 1;
    height: 50px;
}

.flex-150 {
    flex: 1;
    height: 150px;
}

align-content

Esta propiedad solo es útil si tenemos la propiedad flex-wrap con el valor wrap asignado, tenemos múltiples filas y además el contenedor es mas alto que todas las filas.

  • stretch: (Valor por defecto) Estira las filas para ocupar la misma altura.
  • flex-start: Ubica la fila de los elementos en la parte superior del contenedor.
  • flex-end: Ubica la fila en la parte baja del contenedor.
  • center: Centra la fila verticalmente.
  • space-between: La distribuye con espacio equitativo entre filas.
  • space-around: El espacio sobrante es repartido equitativamente a ambos lados.
.flex-container {
    display: flex;
    justify-wrap: wrap;
    height: 300px;
}

.flex {
    flex: 1 0 33%;
}

order

Establece el orden en el cual se mostrarán los elementos.

.flex-1, .flex-2, .flex-3, .flex-4, .flex-5 {
    flex: 1;
}
    
.flex-1 {
    order: 5;
}
 
.flex-2 {
    order: 3;
}

.flex-3 {
    order: 1;
}

.flex-4 {
    order: 2;
}

.flex-5 {
    order: 4;

align-self

Esta propiedad se comporta igual que la propiedad align-items salvo que en este caso aplica solo a un elemento.

.flex-start {
    flex: 1;
    align-self: flex-start;
}

.flex-center {
    flex: 1;
    align-self: center;
}

.flex-end {
    flex: 1;
    align-self: flex-end;
}
