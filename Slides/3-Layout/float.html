float

La propiedad float se encarga se colocar un elemento de forma flotante con respecto a los elementos que tiene a su alrededor. La mejor forma de entenderlo es imaginando una revista, en la que las imágenes se colocan al lado del texto, tanto a la izquierda como a la derecha. Los distintos valores que puede tomar esta propiedad son:

  • left: Desplaza los elementos a la izquierda mientras el contenido se desplaza a lo largo del lado derecho.
  • right: Desplaza el elemento a la derecha y el contenido se desplaza a lo largo de la izquierda.
  • none: Desactiva la propiedad float definida previamente.

Hay que tener en cuenta que las propiedades background y border no se ven afectadas por la propiedad float. Por lo que si un elemento se desplaza al lado de otro elemento float y tiene las propiedades background o border puestas, estas se verán debajo del elemento flotante. Para solucionar este inconviente se le debe aplicar al elemento que contiene el background la propiedad overflow: hidden.

También hay que tener en cuenta que un elemento flotante, lo hace con respecto a los elementos que le siguen a continuación y no a los elementos previos.

.box-left{
    float: left;
    width: 20%;
    height: 75px;
}

.box-right{
    float: right;
    width: 40%;
    height: 50px;
}

clear

Esta propiedad permite que el elemento a continuación de otro flotante no tenga su propiedad float en cuenta. Los valores que puede tomar esta propiedad son:

  • left: Si hay un elemento flotante con valor left anterior no se desplazará a su derecha.
  • right: El mismo comportamiento pero con elementos flotantes con valor right.
  • both: El elemento no se desplaza a un lateral aunque haya elementos flotantes previamente definidos.
  • none: Desactiva la propiedad clear puesta previamente
.box-left {
    float: left;
    width: 200px;
    height: 100px;
}

.box-right {
    float: right;
    width: 100px;
    height: 50px;
}                         

.box-clear-right {
    clear: right;
}

Construir un layout mediante float

Para construir un layout mediante float tenemos dos posibilidades: determinar la ubicación que van a ocupar nuestros elementos y aplicar los estilos float y clear según corresponda. O partir de un grid de columnas y filas y aplicar a cada elemento una clase indicando el número de columnas que queremos que ocupe. El primer caso se muestra en el ejemplo siguiente, el segundo en el apartado Creando un Grid.

.header {
    height: 50px;
}    
                                             
.menu {
    float: left;
    width: 20%;
    height: 400px;
    margin: 5px 5px 5px 0;
}    
                                             
.container {
    float: right;
    height: 400px;                         
    margin-top: 5px;
    overflow: hidden;
}    
                                             
.footer {
    clear: left;
    height: 50px;
}

Problemas de un layout de float

Cuando un elemento se hace flotante, este sale del flujo del documento colocandose encima del resto de elementos y por consiguiente cualquier elemento que se coloque debajo aparecera oculto. Para solucionar este problema se puede realizar los siguiente:

  • Añadir un elemento HTML a continuación asignarle la propiedad clear:both. El problema es que asignamos contenido HTML cuando estamos modificando presentación.
  • Hacer flotante también el elemento oculto. El problema es que pasamos el problema a los siguientes elementos.
  • Utilizar la propiedad overflow:hidden la cual expande el contenido del elemento fuera del elemento flotante.
  • Usar el siguiente estilo en el contenedor de elementos flotantes:
.container::after {
    content: ' ';
    display: table;
    clear: both;
}

Otro problema viene cuando queremos que todos los elementos flotantes tengan la misma altura. En este caso las soluciones pueden consistir en:

  • Añadir un background con una imagen que se repite a lo largo del eje vertical, dando la sensación de que todas las columnas tienen la misma altura.
  • Añadir las columnas dentro de otro elemento y a ese elemento darle un background que ocupe la altura deseada.
