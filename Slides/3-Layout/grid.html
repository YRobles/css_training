Creando un Grid

  • Layout
  • Creando un Grid

La estructura del layout debe componerse de:

  • Contenedor: El contenedor tiene todas las filas.
  • Filas: Fila dentro del contenedor y que contiene las columnas.
  • Columnas: Elemento dentro de la fila y que guarda el contenido.

Los contenedores, filas y columnas solo son elementos de presentacion que nos ayudarán a ubicar nuestro contenido. Dicho contenido de un contenedor propio dentro de una columna, es decir, el contenido no debe ubicarse directamente dentro de la columna sin agruparlo dentro de un padre. Normalmente se trabaja con una reticula de 12 columnas ya que permite la combinacion de mayor numero de multiplos 1, 2, 3, 4 y 6 para distribuir nuestro contenido.

Una distribución HTML de los elementos podría resultar similar a la siguiente:

<div class="container">
    <div class="row">
        <div class="column three"></div>
    </div>             
    <div class="row">
        <div class="column six"></div>
        <div class="column six"></div>
    </div>
</div>

A continuación se muestran los estilos típicos cuando establecemos un grid y como se distribuyen los elementos dentro de dicho grid. Si queremos hacer el grid responsive solo tenemos que añadir las mismas propiedades dentro de las media queries correspondientes.

.row: {
    margin: 0 -5px;
}

.row:after {
    content: '';
    display: table;
    clear: both;
}

[class^=col] {
    float: left;
    padding: 5px;
}
