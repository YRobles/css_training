position

Mediante esta propiedad podemos posicionar el elemento en un determinado lugar, ya sea con respecto al lugar que ocupa por defecto, con respecto al viewport o con respecto a un contenedor padre.

  • static: (Valor por defecto) La propiedad static indica que el elemento debe ocupar la posición que le corresponde según el flujo normal del documento.
  • relative: El elemento se posiciona relativo a la posicion que esta ocupando por defecto. Este valor establece también una referencia para descendientes con posicionamiento absoluto.
  • absolute: El elemento deja de ocupar su posición por defecto y se posiciona con respecto al primer padre con posicionamiento relativo que encuentra o con respecto al <body> en caso de no encontrar ninguno.
  • fixed: El elemento deja de ocupar su posición por defecto y será posicionado con respecto al viewport. Manteniendose siempre fijo aunque se navegue a través del documento.
.box-relative {
    position: relative;
    top: 20px;
    left: 30px;
}

.box-absolute {
    position: absolute;
    bottom: 10px;
    right: 30px;
}

.box-fixed {
    position: fixed;
    top: 50%;
    right: 20px;
}

z-index

Mediante esta propiedad podemos establecer el orden que ocupan las capas una encima de las otras cuando se les aplica un posicionamiento distinto a static.

.box-index-1 {
    top: 130px;
    left: 80px;
    background-color: #29A9DF;
    z-index: 1;
}

.box-index-2 {
    top: 160px;
    left: 30px;
    background-color: #F4792B;
    z-index: 2;
}

.box-index-3 {
    top: 200px;
    left: 100px;
    background-color: #0170BA;
    z-index: 3;
}

    left: 150px;

visibility

Esta propiedad muestra u oculta el elemento. A diferencia de la propiedad display: none, en este caso el espacio que ocupa la etiqueta es reservado.

  • visible: (Valor por defecto) Es el valor normal para todos los elementos. Utizado para devolver a un elemento su visibilidad tras aplicarle una visibilidad hidden.
  • hidden: Oculta el elemento.
.hidden {
    visibility: hidden;
}

opacity

Esta propiedad permite aplicar un nivel de transparencia al elemento y su contenido. Acepta valores decimales entre 0 y 1.

.box-opacity {
    position: relative;
    top: 20px;
    left: -50px;
    background-color: #0170BA;
    opacity: .5;
}
