text-transform

Permite transformar el texto de la siguiente manera:

  • uppercase: Pondrá todas las letras del texto en mayúsculas.
  • lowercase: Pondrá todas las letras del texto en minúsculas.
  • capitalize: Cada palabra empezará con la primera letra en mayúsculas.
p.transform {
    text-transform: uppercase;
}

p.capitalize {
    text-transform: capitalize;
}

line-height

La propiedad line-height permite establecer por cada línea de texto su altura. La propiedad line-height se puede establecer mediante píxeles, porcentaje, ems o incluso no indicar una unidad. En este último caso, el valor será calculado a partir del valor que tiene actualmente el elemento. Esto es útil ya que la propiedad line-height es heredada, de este modo nos saltamos posibles herencias y solo trabajamos con la fuente actual.

.line {
    line-height: 2;
}

Shorthand

Mediante la propiedad font podemos establecer varias propiedades de la fuente de una sola vez. No es necesario indicar todas las propiedades, sin embargo, si es importante que aparezcan en el orden adecuado. Si se indican las propiedades font-size y line-height, estas deben ir separadas por el caracter /. El orden en el que se deben mostrar es: font-style, font-variant, font-weight, font-size, line-height y font-family

.shorthand {
    font: italic  bold small-caps 18px/1.5 Arial, Helvetica, sans-serif;
}
