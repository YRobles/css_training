Soporte por navegador segun fuente

Se permite incluir diferentes formatos de fuente, sin embargo cada uno tiene sus pros y contras (normalmente en el tamaño del fichero) y sobretodo en el soporte que ofrece cada navegador con respecto a un determinado formato. Los formatos disponibles son:

  • .eot: Solo funciona en Internet Explorer.
  • TrueType y OpenType: Fuentes más comunes e instaladas en la mayoria de ordenadores.
  • Web Open Font Format: Formato diseñado especificamente para la web. Se trata de una versión comprimida de TTF y OTF.
  • SVG: No es propiamente un formato de fuente sino el formato vectorial de la misma. No se suele utilizar ya que su tamaños es 2 o 3 veces mayor que el resto.

A continuación se muestra el soporte que ofrecen los navegadores según el formato.

@font-face

Mediante la etiqueta @font-face indicamos que fuentes queremos cargar en nuestra aplicación. Por regla general, conviene disponer de todos los formatos disponibles para que el navegador automaticamente cargue aquel que necesita. No descarga todos, solo coge la primera fuente que encuentra y le funciona correctamente. De este modo, cuando indicamos las fuentes, estas se escriben en el orden que mejor se adapte a nuestra aplicación (normalmente, indicando las fuentes de menor tamaño primero).


@font-face {
    font-family: FontDemo;
    src: url('../assets/fonts/FontDemo.eot');
    src: url('../assets/fonts/FontDemo.eot?#iefix') format('embedded-opentype'),
            url('../assets/fonts/FontDemo.woff2') format('woff2'),
            url('../assets/fonts/FontDemo.woff') format('woff'),
            url('../assets/fonts/FontDemo.ttf') format('truetype'),
            url('../assets/fonts/FontDemo.svg') format('svg');
}

p {
    font-family: FontDemo, sans-serif, monospace;
}

Web Fonts

Variantes en negrita e italica

En vez de tener que dar un nombre a cada fuente segun el estilo, se añaden con las propiedades font-weight y font-style

@font-face {
    font-family: FontDemo;
    src: url('FontDemo.woff2') format('woff2');
    font-weight: normal;
    font-style: normal;
}
            
@font-face {
    font-family: FontDemo;
    src: url('FontDemoItalic.woff2') format('woff2');
    font-weight: normal;
    font-style: italic;
}
                             
@font-face {
    font-family: FontDemo;
    src: url('FontDemoBold.woff2') format('woff2');
    font-weight: bold;
    font-style: normal;
}
                                             
@font-face {
    font-family: FontDemo;
    src: url('FontDemoBoldItalic.woff2') format('woff2');
    font-weight: bold;
    font-style: italic;
}
