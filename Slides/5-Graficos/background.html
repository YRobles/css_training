.bg {

background-image

Mediante esta propiedad establecemos una imagen como fondo del elemento. Hay que tener en cuenta ciertas propiedades que se aplican por defecto y que deben ser indicadas para adaptar la imagen según necesitemos.

¿Cuando aplicamos <img> y cuando background-image?: Si la imagen es simplemente decorativa debemos utilizar background-image. Sin embargo, si la imagen es parte del contenido deberemos utilizar el <img>. Cuando imprimimos una página web, solo las etiquetas <img> aparecerán en la impresión de la página, los background-image serán descartados.

.bg-image {
    background-image: url(../assets/image.png);
}

background-repeat

Mediante esta propiedad podemos modificar el número de apariciones de la imagen dentro del espacio disponible y en que forma lo visualizamos. Los valores que puede tomar esta propiedad son:

  • repeat: (Valor por defecto) Por defecto la imagen se repite tantas veces en horizontal o vertical como el espacio disponible se lo permita.
  • no-repeat: La imagen no se puede repetir.
  • repeat-x: La imagen se repetirá horizontalmente segun el espacio disponible.
  • repeat-y: La imagen se repetirá verticalmente segun el espacio disponible.
  • round: La imagen se repetirá horizontalmente, mientras que verticalmente se estirará hasta ocupar el espacio disponible.
  • space: La imagen se repetirá horizontal o verticalmente tantas veces como el espacio disponible le permita, mostrandose siempre completa.
.bg {
    background-image: url(../assets/image.png);
    background-repeat: round; 
}

background-size

Mediante esta propiedad estamos indicando el tamaño de nuestra imagen, siendo el primer valor su ancho y el segundo su alto. Se puede indicar un valor y dejar el otro en auto, de este modo se respeta la relación de aspecto. Sin embargo, permite otros dos valores que podemos asignar:

  • contain: Fuerza a la imagen a ajustarse al espacio disponible mientras mantiene la relacion de aspecto.
  • cover: Fuerza a ocupar todo el espacio disponible sin respetar la relacion de aspecto.
.bg {
    background-image: url(../assets/image.png);
    background-size: 50px 100px;
}

background-position

Podemos indicar en que coordenadas x e y empezara nuestra imagen. Podemos indicar ambos ejes o de forma independiente mediante las propiedades background-position-x y background-position-y. A parte, esta propiedad incluye los siguientes valores:

  • Horizontal: left | center | right
  • Vertical: top | center | bottom 
.bg {
    background-image: url(../assets/image.png);
    background-position: 30px 20px;
}

background-attachment

Esta propiedad nos permite dejar la imagen fija mientras hacemos scroll.

  • scroll: (Valor por defecto) Al hacer scroll la imagen se desplaza con el resto del contenido.
  • fixed: Mantiene la imagen fija en la posición de la pantalla indicada.
.bg {
    background-image: url(../assets/image.png);
    background-attachment: fixed; 
}

background-origin

Indica donde debe comenzar la imagen en relacion al borde, padding o contenido del elemento. Esta propiedad no afecta si no hay borde y/o padding

  • padding-box: (Valor por defecto) Coloca la imagen con respecto al padding.
  • content-box: Coloca la imagen con respecto al contenido.
  • border-box: Coloca la imagen en la esquina superior izquierda asignada al borde.
.bg-content-origin {
    background-image: url(../assets/image.png);
    background-origin: content-box;
}

.bg-padding-origin {
    background-image: url(../assets/image.png);
    background-origin: padding-box;
}

.bg-border-origin {
    background-image: url(../assets/image.png);
    background-origin: border-box;
}

background-clip

Limita el área donde la imagen puede aparecer.

  • border-box: (Valor por defecto) Permite a la imagen aparecer bajo el contenido debajo de los bordes. Solo visible si los bordes no son sólidos.
  • padding-box: Muestra la imagen dentro del padding y el contenido, pero no debajo de los bordes.
  • content-box: La imagen se muestra solo debajo del contenido.
.bg-content-clip {
    background-image: url(../assets/image.png);
    background-clip: content-box;
}

.bg-padding-clip {
    background-image: url(../assets/image.png);
    background-clip: padding-box;
}

.bg-border-clip {
    background-image: url(../assets/image.png);
    background-clip: border-box;
}

Shorthand

La propiedad background es un shorthand que nos permite definir a la vez distintas propiedades del background. Si se indican las propiedades background-position y background-size se deben separar con /.

.bg {
    background: url(../assets/image.png) left top/20px 30px repeat scroll padding-box; 
}
