Gradiente lineal

.gradient-1 {
    background-image: linear-gradient(to right, #0170BA, white);
}

.gradient-2 {
    background-image: linear-gradient(135deg, white, #0170BA);
}

.gradient-3 {
    background-image: linear-gradient(to right, white, #0170BA 30%, #0170BA 70%, white);
}

    background-image: linear-gradient(to right, black, transparent), 
                                       linear-gradient(100deg, magenta, transparent), 
                                       linear-gradient(40deg, yellow, transparent);
}

repeating-linear-gradient

.bg {
    background-image: repeating-linear-gradient(45deg, #0170BA 20px, #29A9DF 30px, #0170BA 40px);
}

Gradiente radial

El origen del centro del gradiente se puede también especificar mediante los siguientes valores:

  • closest-side: Extiende el gradiente hasta el lado mas cercano.
  • closest-corner: Lo extiende hasta la esquina mas cercana.
  • farthest-side: Hasta el lado mas alejado.
  • farthest-corner: Hasta la esquina más alejada.
.gradient-radial-1 {
    background-image: radial-gradient(circle, red, blue);
}
    
.gradient-radial-2 {
    background-image: radial-gradient(circle at 20%, red, blue);
}

repeating-radial-gradient

.bg {
    background-image: repeating-radial-gradient(circle, #0170BA 20px, #FFF 40px, #0170BA 50px);
}
