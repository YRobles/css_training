Sombras

  • Gráficos y animaciones
  • Sombras

Cuando definimos la sombra de un elemento hay 4 valores en los que se descompone que debemos tener en cuenta:

  • Desplazamiento horizontal.
  • Desplazamiento vertical.
  • Amplitud de la sombra.
  • Distancia desde la caja donde queremos que empiece a mostrarse la sombra.

box-shadow

.element {
    box-shadow: 0 0 10px black;
}

sombra interna

.element {
    box-shadow: inset 0 0 10px black;
}

Expandiendo la sombra

.element {
    box-shadow: 2px 2px 5px 5px black;
}

Combinando sombras

.element {
    box-shadow: inset 0 0 10px black, 0 0 10px orange, 2px 2px 5px black;
}
