Transformaciones

  • Gráficos y animaciones
  • Transformaciones

Las transformaciones nos permiten manipular un elemento tanto en 2D como en 3D. Las principales transformaciones que podemos realizar son: rotación, escalado, traslación y sesgado. Para las propiedades de rotación y sesgado el valor debe ser asignado en grados (deg).

rotate

.rotate {
    transform: rotate(90deg);
}

.rotate-x {
    transform: rotateX(60deg);
}

.rotate-y {
    transform: rotateY(120deg);
}

.rotate-z {
    transform: rotateZ(45deg);
}

scale

.item {
    transform: scale(2);
}

.item {
    transform: scale(1.5);
}

.item {
    transform: scale(1.75, .5);
}

.item {
    transform: scaleX(1.5);
}

.item {
    transform: scaleY(1.75);
}

translate

.item {
    transform: translate(20px, 10px);
}

.item {
    transform: translateX(20px);
}                 

.item {
    transform: translateY(10px);
}

skew

.skew {
    transform: skew(-30deg, 0);
}

.skew-x {
    transform: skewX(60deg);
}

.skew-y {
    transform:  skewY(45deg);
}

transform-origin

Esta propiedad indica desde que punto se debe tomar como centro de gravedad para realizar la transformación.

.item {
    transform-origin: left top;
    transform: rotateZ(-15deg);
}
