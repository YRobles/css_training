Transiciones

  • Gráficos y animaciones
  • Transiciones

Las transiciones son animaciones que se producen automaticamente entre un estado inicial y el estado final. A diferencia de las animaciones, no podemos crear o modificar fotogramas clave.

transition-property y transition-duration

Estas dos propiedades son las mínimas que debemos indicar para que se produzca una transición. La primera indica a que propiedades afecta y la segunda el tiempo que durará la transición. Las duraciones se pueden indicar en segundos (s) o milisegundos (ms).

.box{
    left: 0;
    width: 100px;
    background-color: #0170BA;
    transition-property: left, background-color;
    transition-duration: 2s, 1s;
}
                                             
.container:hover .box{
    left: calc(100% - 100px);
    background-color: #AAA;
}

transition-timing-function

Indica la velocidad a la que se producirá la animación. Consta de una serie de valores predefinidos, sin embargo, a parte de los ya predefinidos se puede establecer una curva de Bezier: cubic-bezier(...)

.box{
    left: 0;
    width: 100px;
    background-color: #0170BA;
    transition-property: left, background-color;
    transition-duration: 2s, 1s;
    transition-timing-function: ease-in-out;
}
                                             
.container:hover .box{
    left: calc(100% - 100px);
    background-color: #AAA;
}

transition-delay

Indica un pequeño retraso hasta que comience la transición.

.box{
    left: 0;
    width: 100px;
    background-color: #0170BA;
    transition-property: left, background-color;
    transition-duration: 2s, 1s;
    transition-delay: 1s, .5s;
}
                                             
.container:hover .box{
    left: calc(100% - 100px);
    background-color: #AAA;
}

Shorthand

Podemos indicar todas las propiedades afectas a la vez mediante esta propiedad.

.box{
    left: 0;
    width: 100px;
    background-color: #0170BA;
    transition: all 2s 300ms ease-in-out;
}
                                             
.container:hover .box{
    left: calc(100% - 100px);
    background-color: #AAA;
}
