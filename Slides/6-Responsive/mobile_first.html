Mobile First vs Desktop First

  • Diseño responsive
  • Mobile First vs Desktop First

No hay que diseñar un estilo por cada resolucion o dispositivo, se debe empezar con un estilo y a partir de ahí irlo adaptando por medio de media queries

  • Desktop first: Se diseña el sitio con la visualización para escritorio por defecto. Despues se añaden las media queries para adaptarlo a tables primero y a moviles después. Las modificaciones consistirán en ir eliminando columnas, haciendo el texto más pequeño, etc.

  • Mobile First: Se diseña primero para movil y despues se añaden media queries que van refinando el diseño para tablet primero y despues para escritorio. Se irán añadiendo columnas y ajustando el contenido al nuevo tamaño disponible.
