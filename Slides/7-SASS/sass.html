Introducción a SASS

  • Introducción a SASS

SASS es un precomilador de CSS que nos permite escribir código CSS de una manera más sencilla. Para ello, este precompilador nos permite añadir funcionalidades de las que CSS no consta, como creación de variables, anidamientos, herencia o mixins. SASS incluye muchas otras funcionalidades, como crear funciones o bucles, sin embargo estas primeras son las más comunmente utilizadas.

Variables

Mediante CSS no podemos crear variables, lo cual puede resultar costoso cuando hay que aplicar el mismo cambio a distintos estilos a lo largo de las hojas de estilos. Por ejemplo, porque hemos modificado los colores de nuestra página web.


$font_medium: 18px;
$blue_dark: #0170BA;            

.variables {
    font-size: $font_medium;
    color: $blue_dark;
}

Anidamiento

En CSS no se permite el anidamiento, es decir, si queremos aplicar un estilo a partir de los padres del elemento, debemos escribir todos los elementos de ese anidamiento. Mediante SASS, podemos establecer una jerarquia de estilos que aplicarán al elementos y sus descendientes. Aunque está funcionalidad facilita mucho el trabajo, debemos ser conscientes de que tiende a un abuso de selectores de etiquetas cuando generemos los estilos, estás etiquetas provocarán una mayor especificidad del estilo, siendo en ocasion bastante dificil de sobreescribir según el nivel de anidamiento.

.container {
    .content {
        .element {
            width: 100px;
            height: 100px;
            background-color: $blue_dark;
        }
    }
}

Herencia

Mediante la herencia podemos aplicar a un estilo todas las propiedades de otro estilo anterior, sin tener que repetir todas y cada una de ellas para el nuevo estilo.

.box-1 {
    width: 100px;
    height: 100px;
    margin: 10px;
    background-color: $blue_dark;
}
.box-2 {
    @extend .box-1;
    box-shadow: 3px 3px 5px #000;
}
.box-3 {
    @extend .box-2;
    margin-left: 50px;
}

Mixins

Los mixins son pequeños atajos que nos permiten definir una vez un conjunto de propiedades que se repiten muchas veces mediante una única llamada.

@mixin cross-transform( $value) {
    _webkit-transform: $value;
    _ms-transform: $value;
    _moz-transform: $value;
    _o-transform: $value;
    transform: $value;
}                     

.transform {
    @extend .box-1;
    @include transform(rotate(45deg))
}
